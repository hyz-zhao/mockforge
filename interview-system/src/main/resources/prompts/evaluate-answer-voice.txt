# 评分标准（务必严格遵守，宁严勿宽）

## 零分/极低分判定（必须严格执行）
- 回答少于10个字且无任何技术关键词 → 直接给 0 分
- 回答少于30字且未提到任何相关知识点 → 给 0-5 分
- 回答"不知道"、"不会"、"不了解"等 → 直接给 0 分
- 回答与问题完全无关 → 直接给 0 分
- 仅重复问题中的关键词但无解释 → 给 0-5 分

## 内容评分（accuracy + depth）
- 仅提到1-2个关键词但无任何解释 → 给 5-15 分
- 概念错误或表述完全错误 → 给 0-10 分
- 回答笼统模糊、泛泛而谈 → 给 10-20 分

## 语音表达评分（fluency + nervousness）
根据转文字内容判断语音表达质量：
- 流畅度（fluency）：
  - 文字中大量出现"嗯"、"啊"、"那个"、"就是"等口头禅 → fluency 给 0-2 分
  - 句子断断续续、逻辑不连贯 → fluency 给 1-3 分
  - 表达基本流畅但偶有停顿 → fluency 给 4-6 分
  - 表达流畅、逻辑清晰 → fluency 给 7-10 分

- 紧张程度（nervousness）：
  - 大量重复词语、语无伦次、频繁自我纠正 → nervousness 给 0-2 分（非常紧张）
  - 偶尔结巴、有些犹豫 → nervousness 给 3-5 分（比较紧张）
  - 基本自然、偶有紧张 → nervousness 给 6-8 分（轻微紧张）
  - 表达自信、从容不迫 → nervousness 给 9-10 分（不紧张）

## 分数档位参考
- 0分：完全不会、答非所问、回答过短无内容
- 1-10分：仅提到极少量关键词，几乎无解释，语音表达极差
- 11-25分：提到部分关键词，解释非常浅显，语音表达较差
- 26-40分：覆盖了少量要点，但解释不够，语音表达一般
- 41-60分：覆盖了大部分要点，解释基本正确，语音表达尚可
- 61-80分：覆盖了所有要点，解释准确，语音表达流畅
- 81-100分：完美回答，覆盖所有要点且有深入理解，语音表达自信流畅

## 各维度评分细则
- accuracy（准确性）：核心概念是否正确。概念错误直接给 0-1 分
- depth（深度）：是否深入理解原理。仅提关键词给 0-1 分
- fluency（流畅度）：语音表达是否流畅。大量口头禅给 0-2 分
- nervousness（紧张度）：是否紧张。语无伦次给 0-2 分

## 重要提醒
- 不要因为是模拟面试就放宽标准
- 宁可给0分让候选人知道自己不足，也不要给高分误导
- 如果参考答案有5个要点，候选人只提到1个，overallScore 不应超过 15 分
- 回答中如果有任何明显的概念错误，overallScore 不应超过 20 分
- 语音模式下，内容质量和语音表达各占50%权重
